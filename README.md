# Personal Budget Analyzer using Python

## Objective
A Python-based personal finance project that tracks monthly income, expenses, total expenses, and savings.

## Tools Used
- Python
- Pandas
- Matplotlib
- CSV file handling

## Features
- Add monthly budget data
- Store data in CSV
- View saved data
- Visualize monthly savings
- Compare income and expenses
- Analyze expense categories

## Skills Demonstrated
- While loops
- Dictionaries
- Pandas DataFrame
- CSV storage
- Data visualization
- Financial calculations

## Output
The project generates charts such as monthly savings and income vs expenses.
